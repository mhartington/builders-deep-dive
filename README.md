# How to use this

## custom-builders dir

cd into `custom-builders`, install deps, build, and link.

```bash
cd custom-builders
npm install
npm run build
npm link
```

## demo-app dir

cd into `demo-app`, install deps, and link `@bionic/builders`


```bash
cd demo-app
npm install
npm link @bionic/builders
```


That should all you need!
